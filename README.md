# Novel-Editor 小说编辑器

electron+typescript-vue     项目新建于： @quick-start/electron

| 渲染进程下的技术栈 | electron     |
| ------------------ | ------------ |
| 基础框架           | vue          |
| 状态管理           | pinia        |
| 组件库             | element-puls |



### 安装依赖

```bash
$ npm i        // 用yarn的话会出现依赖安装失败的情况
```

### 开发环境启动

```bash
$ npm dev
```

### 打包

```bash
# For windows
$ npm build:win

# For macOS
$ npm build:mac

# For Linux
$ npm build:linux
```
